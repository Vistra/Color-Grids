# C o l o r G r i d s

About

ColorGrids is an HSL based color selection tool aimed at helping designers experiment with, and create interesting color combinations. ColorGrids makes it easy to select colors that are mathematically related to each other within the HSL colorspace.

Use the sliders below to define your grids, then start selecting colors.

Or try hitting randomize to roll the dice for a new color palette.
